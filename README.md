# 音声文字起こしアプリケーション
## 概要
- このアプリケーションは、ユーザーが録音した音声を文字起こしし、テキストファイルに保存するためのものです。

## 必要環境
```
python 3.13.7

ffmpeg
PySide6
mlx>=0.11
numba
numpy
torch
tqdm
more-itertools
tiktoken
huggingface_hub
scipy

whisper-base-mlx
whisper-large-v3-turbo
```

## インストール手順
1. 必要なライブラリをインストールします。
    ```bash
    pip install -r requirements.txt
    ```
2. アプリケーションを実行します。
    ```bash
    python main.py
    ```

## 使用方法
1. アプリケーションを起動します。
2. 「録音開始」ボタンをクリックして、音声を録音します。
3. 録音が終了したら、「文字起こし」ボタンをクリックします。
4. 録音した音声が文字起こしされ、「テキストファイルに保存」ボタンをクリックして、テキストファイルに保存します。

## 注意事項
- このアプリケーションは、ユーザーが録音した音声を文字起こしし、テキストファイルに保存するものです。
- 録音する前に、マイクが正しく接続されていることを確認してください。
- 録音した音声は、アプリケーションのディレクトリーに保存されます。
