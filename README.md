# 音声文字起こしアプリケーション
## 概要
- このアプリケーションは、ユーザーが録音した音声を文字起こしし、テキストファイルに保存するためのものです。

## 必要環境
```
python 3.13.7

ffmpeg
PySide6
mlx>=0.11
numba
numpy
torch
tqdm
more-itertools
tiktoken
huggingface_hub
scipy

whisper-base-mlx
whisper-large-v3-turbo
```
